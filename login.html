変数（へんすう） - JavaScript の基本

変数とは、値を保存する「箱」のようなものです。

1. 変数の宣言方法

JavaScript では、変数を宣言する方法が 3 つあります。

宣言方法 読み方 使う場面 特徴 例
var ヴァー 古い書き方（ES5以前） 関数スコープ・hoistingあり var x = 10;
let レット 最も一般的 ブロックスコープ（安全） let age = 18;
const コンスト 値を固定したいとき 再代入できない（定数） const name = "Aoi";
🔹 現在は let と const を使うのが一般的です。

2. データ型（Data Types）

変数にはさまざまな種類の値を入れることができます。

データ型 例 説明
Number（数値） let score = 99; 整数・小数などの数
String（文字列） let name = "Aoi"; 文字・文章を表す
Boolean（真偽値） let pass = true; true または false
Undefined let x; 値がまだ代入されていない
Null let y = null; 意図的に「空」にする
Object（オブジェクト） let person = {name:"Aoi", age:18}; 複数の値をまとめて保持
Array（配列） let fruits = ["apple","banana"]; 複数の値を順番に保持
Symbol let id = Symbol("id"); 一意な値を表す
BigInt let big = 1234567890123456789n; 非常に大きな数値を扱う

3. 使用例

              
                let age = 20;                // 数値
                         const name = "Aoi";         // 文字列（定数）
                 let pass = true;            // 真偽値
                let colors = ["red","blue"]; // 配列
                              let person = {name:"Aoi", age:20}; // オブジェクト
               let nothing = null;         // Null
                    let something;              // Undefined

4. まとめ

種類 説明 再代入できる？
let 一般的な変数（何でも入る） ✅ できる
const 定数（値を固定） ❌ できない
var 古い書き方（非推奨） ✅ できるが注意
📝 このページは JavaScript の 変数（Variables） の基礎をまとめたものです。
他の例（配列・オブジェクト・スコープなど）も追加できます。

データ型（Data Types） - JavaScript の基本

JavaScript では、変数にいろいろな種類の値を入れることができます。

1. データ型とは？

データ型（Data Type）とは、変数の中にどんな種類の値が入っているかを示すものです。 JavaScript では「プリミティブ型」と「オブジェクト型」に分かれます。

🔹 プリミティブ型：1つの値だけを持つ（例：数値、文字列、真偽値など）
🔹 オブジェクト型：複数の値をまとめて扱う（例：配列、オブジェクトなど）

2. プリミティブ型（Primitive Types）

データ型 例 説明
Number（数値） let age = 20; 整数・小数・負の数など
String（文字列） let name = "Aoi"; 文字や文章。「"」または「'」で囲む
Boolean（真偽値） let pass = true; true（真）か false（偽）
Undefined let x; まだ値が代入されていない状態
Null let y = null; 「何もない」ことを明示的に示す
Symbol let id = Symbol("id"); 一意（ユニーク）な値を作る
BigInt let big = 9007199254740993n; 非常に大きな整数を扱える

3. オブジェクト型（Object Types）

複数の値やデータをまとめて管理できる型です。

型 例 説明
Object（オブジェクト） let person = { name:"Aoi", age:18 }; キーと値のペアをまとめる
Array（配列） let colors = ["red","blue","green"]; 複数の値を順番に格納
Function（関数） function sayHi(){ alert("こんにちは"); } 動作（処理）をまとめたもの
Date let now = new Date(); 日付や時刻を扱う

4. typeof 演算子で型を調べる


                    let x = 10;
                       let y = "Aoi";
                      let z = true;
      
                                           console.log(typeof x); // "number"
                                           console.log(typeof y); // "string"
                                            console.log(typeof z); // "boolean"
💡 typeof は値の型を文字列で返します。

5. まとめ

分類 型の名前 例
プリミティブ型 Number, String, Boolean, Undefined, Null, Symbol, BigInt 単一の値を保持
オブジェクト型 Object, Array, Function, Date など 複数の値や構造を保持
📝 このページは JavaScript の データ型 (Data Types) の基礎をまとめたものです。
もし希望があれば、演習付き（型を自動判別する入力フォーム）も作成できます。

演算子（Operators） - JavaScript の基本

JavaScript では、計算・比較・代入などを行うために「演算子」を使います。

1. 演算子とは？

演算子（Operator）とは、値や変数に対して計算・比較・処理を行うための記号のことです。

🔹 算術演算子：足す、引く、掛けるなどの計算
🔹 比較演算子：値の大小や一致を調べる
🔹 論理演算子：条件を組み合わせる
🔹 代入演算子：変数へ値を入れる・更新する

2. 算術演算子（Arithmetic Operators）

演算子 例 説明
+ 5 + 3 足し算
- 7 - 2 引き算
* 4 * 6 掛け算
/ 10 / 2 割り算
% 10 % 3 余り（モジュロ）
** 2 ** 3 べき乗（Power）

3. 比較演算子（Comparison Operators）

演算子 例 説明
== 5 == "5" 値が等しい（型は無視）
=== 5 === 5 値と型が等しい
!= 5 != 3 値が異なる
!== 5 !== "5" 値または型が異なる
> 7 > 2 より大きい
< 3 < 5 より小さい
>= 5 >= 5 以上
<= 3 <= 4 以下

4. 論理演算子（Logical Operators）

演算子 例 説明
&& true && false AND（両方真なら真）
|| true || false OR（どちらか真なら真）
! !true NOT（真偽を反転）

5. 代入演算子（Assignment Operators）

演算子 例 説明
= x = 5 代入
+= x += 1 x に 1 を足す
-= x -= 1 x から 1 を引く
*= x *= 2 x を 2 倍
/= x /= 2 x を半分に
%= x %= 3 余りを代入

6. typeof を使って演算子の結果を調べる


          let a = 10 + 5;
          let b = 10 > 3;
                 let c = true && false;
          
                             console.log(typeof a); // "number"
                              console.log(typeof b); // "boolean"
                              console.log(typeof c); // "boolean"
💡 typeof は値の型を文字列で返します。

7. まとめ

分類 演算子 説明
算術演算子 +, -, *, /, %, ** 計算を行う
比較演算子 ==, ===, !=, !==, >, <, >=, <= 値の比較を行う
論理演算子 &&, ||, ! 条件を組み合わせる
代入演算子 =, +=, -=, *=, /=, %= 変数へ値を代入・更新
📝 このページは JavaScript の 演算子 (Operators) の基礎をまとめたものです。
必要なら、演習問題・自動チェック機能も追加できます。

制御構文（Control Flow） - JavaScript の基本

プログラムを「どの順番で実行するか」をコントロールする仕組みです。

1. 制御構文とは？

制御構文（Control Flow）とは、条件分岐や繰り返しなど、プログラムの流れを制御するための構文です。

🔹 if / else：条件で処理を変える
🔹 switch：複数の選択肢から選ぶ
🔹 for / while：処理を繰り返す
🔹 break / continue：ループ制御

2. 条件分岐（if / else）

構文 説明
if (条件) { ... } 条件が true のとき実行
else { ... } 上の条件が false のとき実行
else if (条件) { ... } 条件を追加 

          // 例：スコア判定
       let score = 85;
          
          if (score >= 90) {
                      console.log("Aランク");
                  } else if (score >= 70) {
                       console.log("Bランク");
 } else {
                       console.log("Cランク");
    }

3. switch 文（複数分岐）

同じ変数を複数の値で分岐するときに便利。

キーワード 説明
case 条件に一致したときの処理
break switch 文を終了する
default どれにも一致しなかったとき 

              let fruit = "apple";
          
          switch (fruit) {
       case "apple":
                    console.log("りんごです");
break;
        case "banana":
                    console.log("バナナです");
break;
  default:
                    console.log("不明な果物");
          }

4. 繰り返し（Loops）

構文 説明
for 決まった回数ループ
while 条件が true の間ループ
do...while 少なくとも1回実行してからループ

▶ for 文の例


            for (let i = 1; i <= 5; i++) {
                 console.log(i);  // 1〜5 を順に表示
}

▶ while 文の例


let n = 3;
     while (n > 0) {
           console.log(n);
n--;
}

▶ do...while の例


      let x = 0;
do {
                                 console.log("1回は実行される");
      x++;
            } while (x < 0);

5. break / continue

キーワード 説明
break ループを強制終了
continue 次のループにスキップ

▶ break の例


                       for (let i = 1; i <= 10; i++) {
                if (i === 5) break;
                             console.log(i); // 1〜4 で止まる
}

▶ continue の例


                       for (let i = 1; i <= 5; i++) {
                    if (i === 3) continue;
                                            console.log(i); // 1, 2, 4, 5 （3 をスキップ）
}

6. まとめ

分類 構文 説明
条件分岐 if / else, else if 条件に応じて処理を分ける
複数分岐 switch 複数の値に応じた処理
繰り返し for, while, do...while ループ処理を行う
ループ制御 break, continue ループの動きを調整
📝 このページは JavaScript の 制御構文（Control Flow） の基礎をまとめたものです。
もし希望があれば、練習ページ・問題集・自動採点版も作成できます。

関数（Function） - JavaScript の基本

関数とは、特定の処理をまとめた「再利用可能な命令の塊」です。

1. 関数とは？

JavaScript の関数は、同じ処理を何度も使いたいときや、処理を整理したいときに使います。

特徴 説明 例
再利用可能 一度作れば何度でも呼び出せる greet("Alice");
入力と出力 パラメータを受け取り、値を返すことができる return a + b;
整理しやすい コードのまとまりを作って見やすくできる function add(a, b) { ... }
🔹 関数はプログラミングの基本で、オブジェクトや配列と一緒に使うことが多いです。

2. 関数の基本構文

function キーワードで作成し、return で値を返します。

              
      function greet(name) {
                    return "Hello, " + name + "!";
}
          
                               console.log(greet("Alice")); // "Hello, Alice!"
操作 説明 例
呼び出し 関数の名前に引数を渡して実行 greet("Bob");
値を返す return を使って計算結果を返す let sum = add(2, 3);
パラメータなし 引数なしで呼び出せる関数も作れる sayHi();

3. 関数の種類

関数にはいくつかの書き方があります。

種類 説明 例
名前付き関数 function キーワードで定義 
function add(a, b) { return a + b; }
匿名関数 変数に代入して使う 
const multiply = function(a, b) { return a * b; };
アロー関数 ES6以降、短く書ける 
const divide = (a, b) => a / b;

4. 関数とオブジェクトの組み合わせ

オブジェクトの中に関数を入れると「メソッド」と呼びます。

              
          const dog = {
              name: "ポポ",
                   bark: function() {
                     return "ワン！"
  }
};
          
                                 console.log(dog.bark()); // "ワン！"

5. まとめ

機能 説明
再利用 同じ処理を何度も使える
パラメータ 入力値を受け取ることができる
戻り値 計算結果などを返せる
オブジェクト内で使用 メソッドとして関数を持たせられる
📝 このページは JavaScript の 関数（Functions） の基礎をまとめたものです。
オブジェクト・配列・メソッドなどと組み合わせて、より複雑な処理を作ることができます。

オブジェクト（Object） - JavaScript の基本

オブジェクトとは、複数の値を「キー : 値」の形でまとめて管理できるデータ構造です。

1. オブジェクトとは？

JavaScript のオブジェクトは、1つの名前の中に複数の情報をまとめたいときに使います。

特徴 説明 例
キーと値 「name: 'Aoi'」のように、名前と値のペアで構成される { name: "Aoi", age: 18 }
種類をまとめられる 文字列・数値・関数など、いろいろ入れられる person.name / person.age
柔軟性が高い あとから項目を追加・削除できる person.city = "Tokyo";
🔹 オブジェクトは JavaScript を使う上で最も重要なデータ構造の一つです。

2. オブジェクトの基本構文

キーと値の組み合わせで作ります。

              
          const person = {
            name: "Aoi",
        age: 18,
             city: "Tokyo"
};
操作 説明 例
値の取得 ドット（.）またはブラケットでアクセス person.name
値の追加 新しいキーと値を追加できる person.country = "Japan";
値の削除 delete を使用 delete person.city;

3. メソッド（オブジェクトの関数）

オブジェクトの中に関数を入れることもできます。

              
          const dog = {
               name: "ポポ",
                    bark: function() {
                       return "ワン！";
        }
};
          
                                 console.log(dog.bark()); // "ワン！"

4. オブジェクトと配列の組み合わせ

現実のデータは、よく「配列 + オブジェクト」で表現されます。

              
          const students = [
                         { name: "Aoi", age: 18 },
                         { name: "Sora", age: 17 }
];
用途 説明
リスト表示 ユーザー一覧、商品一覧などに使う
データの管理 1つの項目に複数の情報がある時に便利

5. まとめ

機能 説明
キーと値のセット 複数の情報を1つにまとめられる
柔軟性 追加・変更・削除が簡単
メソッド 関数を持つことができる
配列との相性 オブジェクトの配列は実務で最もよく使われる形
📝 このページは JavaScript の オブジェクト（Objects） の基礎をまとめたものです。
関数・クラス・プロトタイプなど、さらに詳しく説明を追加できます。

配列（Array） - JavaScript の基本

配列とは、複数の値をまとめて順番に保存できるデータ構造です。

1. 配列とは？

配列（Array）は、複数のデータを1つの変数で管理するための仕組みです。

特徴 説明 例
複数の値を保存 順番に並んだデータをまとめて扱う ["red","blue","green"]
インデックスでアクセス 0番から始まる番号で取り出す colors[0]
柔軟に変更可能 追加・削除・変更が簡単 colors.push("yellow")
🔹 配列は JavaScript の中で最もよく使われるデータ構造の1つです。

2. 配列の作り方と基本操作

代表的な操作をまとめています。

              
                      const colors = ["red", "blue", "green"];
          
                     console.log(colors[0]); // red（0番目）
                      console.log(colors.length); // 長さ（3）
操作 説明 例
要素を追加 末尾に追加 colors.push("yellow");
要素を削除 最後を削除 colors.pop();
先頭に追加 最初に追加される colors.unshift("black");
先頭を削除 最初の要素が消える colors.shift();
要素の変更 インデックスを指定して変更 colors[1] = "pink";

3. 配列のループ（繰り返し）

配列は繰り返し処理と相性がとても良いです。

              
                                const fruits = ["apple", "banana", "orange"];
          
              for (let fruit of fruits) {
            console.log(fruit);
    }
// apple
// banana
// orange

4. 配列の便利メソッド

よく使うメソッドをまとめています。

メソッド 説明 例
map() 全ての要素を変換して新しい配列を返す nums.map(n ⇒ n * 2)
filter() 条件に合うものだけ取り出す nums.filter(n ⇒ n > 10)
find() 最初に一致した要素を返す users.find(u ⇒ u.id === 1)
includes() 値が含まれているか調べる colors.includes("red")
join() 配列を文字列に変換 ["a","b"].join("-")

5. 配列とオブジェクトの組み合わせ

実務では最も多い形です。

              
          const users = [
                             { name: "Aoi", age: 18 },
                              { name: "Sora", age: 17 },
                             { name: "Yuki", age: 19 }
  ];
          
                            console.log(users[0].name); // Aoi

6. まとめ

ポイント 説明
複数の値を管理 データをまとめて扱える
柔軟に操作可能 追加・削除・変更が簡単
メソッドが豊富 処理の効率が高い
実務で必須 API・データ管理で必ず使う
📝 このページは JavaScript の 配列（Arrays） の基礎をまとめたものです。
必要であれば、map/filter の練習問題やビジュアル解説も追加できます。

メソッド（Method） - JavaScript のオブジェクトの関数

メソッドとは、オブジェクトの中に定義された関数のことです。

1. メソッドとは？

メソッドは、オブジェクトに関連する処理をまとめるための関数です。

特徴 説明 例
オブジェクトに属する オブジェクトのプロパティとして関数を持つ dog.bark();
this キーワード 同じオブジェクト内のプロパティにアクセスできる this.name
再利用可能 オブジェクトのデータに基づいて複数回呼び出せる user.sayHi();

2. メソッドの基本例

              
  const dog = {
        name: "ポポ",
             bark: function() {
                                       return this.name + "はワン！と鳴く";
    }
};
          
                                     console.log(dog.bark()); // "ポポはワン！と鳴く"

3. 短縮記法（ES6）

ES6 以降は function キーワードを省略してメソッドを定義できます。

              
          const cat = {
                   name: "ミケ",
              meow() {
                                                      return this.name + "はニャー！と鳴く";
      }
};
          
                                               console.log(cat.meow()); // "ミケはニャー！と鳴く"

4. メソッドと配列の組み合わせ

オブジェクトの配列内にメソッドを持たせることも可能です。

              
      const users = [
  {
                 name: "Aoi",
                                                   greet() { return "こんにちは、" + this.name; }
    },
  {
                  name: "Sora",
                                                   greet() { return "こんにちは、" + this.name; }
}
];
          
                                           console.log(users[0].greet()); // "こんにちは、Aoi"
                                            console.log(users[1].greet()); // "こんにちは、Sora"

5. まとめ

機能 説明
オブジェクト内の関数 そのオブジェクト専用の処理をまとめる
this キーワード 自分自身のプロパティにアクセスできる
再利用可能 オブジェクトのデータに応じて複数回呼び出せる
配列との組み合わせ オブジェクト配列の各要素にメソッドを持たせられる
📝 このページは JavaScript の メソッド（Methods） の基礎をまとめたものです。
オブジェクト・関数・配列と組み合わせて、より実務的なデータ操作が可能です。

JavaScript のメソッド（Methods）一覧

JavaScript の各データ型で使える便利なメソッドをまとめています。

1. 文字列（String）のメソッド

メソッド 説明 例
charAt() 指定位置の文字を返す "Hello".charAt(1); // "e"
charCodeAt() 指定位置の文字コードを返す "A".charCodeAt(0); // 65
concat() 文字列を結合 "Hello".concat(" World"); // "Hello World"
includes() 文字列に指定文字が含まれるか "Hello".includes("e"); // true
indexOf() 指定文字の最初の位置を返す "Hello".indexOf("l"); // 2
lastIndexOf() 指定文字の最後の位置を返す "Hello".lastIndexOf("l"); // 3
match() 正規表現にマッチする文字列を返す "abc123".match(/\d+/); // ["123"]
replace() 文字列を置換 "Hello".replace("H","J"); // "Jello"
search() 正規表現にマッチする位置を返す "abc123".search(/\d/); // 3
slice() 文字列を切り出す "Hello".slice(1,4); // "ell"
split() 文字列を配列に分割 "a,b,c".split(","); // ["a","b","c"]
substring() 文字列を切り出す（slice類似） "Hello".substring(1,4); // "ell"
toLowerCase() 小文字に変換 "HELLO".toLowerCase(); // "hello"
toUpperCase() 大文字に変換 "hello".toUpperCase(); // "HELLO"
trim() 前後の空白を削除 " hello ".trim(); // "hello"
startsWith() 指定文字で始まるか判定 "Hello".startsWith("He"); // true
endsWith() 指定文字で終わるか判定 "Hello".endsWith("lo"); // true
repeat() 文字列を繰り返す "ha".repeat(3); // "hahaha"

2. 配列（Array）のメソッド

メソッド 説明 例
push() 配列の末尾に追加 [1,2].push(3); // [1,2,3]

DOM（Document Object Model） - JavaScript の基本

DOM は、HTML を「木構造のオブジェクト」として扱う仕組みです。

1. DOMとは？

DOM（Document Object Model）とは、HTML や XML をブラウザが読み込んだときに生成される 「木構造のオブジェクト」のことです。JavaScript を使ってページの内容を読み取ったり、 書き換えたり、追加したりできます。

要素 読み方 説明

Element エレメント <div> や <p> などの HTML 要素
Node ノード DOM を構成する基本要素（テキストも含む）
Attribute アトリビュート id、class など要素の属性
Event イベント click, input などのユーザー操作
🔹 DOM を使うと、HTML の内容を JavaScript で自由に操作できます。

querySelectorAll() クエリ セレクター オール 一致するすべての要素を取得 document.querySelectorAll("p");

操作 説明 例

非同期処理（Asynchronous） & API - JavaScript の基本

非同期処理とは、時間のかかる処理を待たずに他の処理を進める仕組みです。

1. 非同期処理とは？

JavaScript は「シングルスレッド」で動作しますが、非同期処理 を使うことで 時間のかかる操作（通信・タイマー・ファイル読み込みなど）の完了を待たずに、 他の処理を同時に進められます。

種類 読み方 説明
Callback コールバック 関数を引数として渡し、完了したら実行
Promise プロミス 非同期結果を扱うオブジェクト
async / await アシンク / アウェイト Promise を簡単に書く文法
🔹 現在は async / await がもっとも一般的です。

2. APIとは？

API（エーピーアイ）とは、データを取得したり送信したりするための「外部サービスとの入り口」のことです。 JavaScript では fetch() を使って API にアクセスします。

用語 読み方 説明
API エーピーアイ アプリ同士がデータをやり取りする仕組み
HTTP エイチ ティー ティー ピー Web の通信ルール（GET, POST など）
Endpoint エンドポイント API を呼び出す URL のこと
JSON ジェイソン 最も一般的なデータ形式

3. Promise と async/await の比較

種類 説明 例
Promise .then() と .catch() を使う fetch(url) .then(res => res.json()) .then(data => console.log(data));
async / await 同期処理のように書けて読みやすい const data = await fetch(url).then(r => r.json()); console.log(data);
🔹 新しいコードは async/await を使うのが基本です。

4. API の使用例（fetch）

              
                // API からデータを取得する例
                async function getUser() {
                                                   const url = "https://jsonplaceholder.typicode.com/users/1";
                                                  const response = await fetch(url);   // データ取得
                                                   const data = await response.json();  // JSON に変換
                                                   console.log(data.name);              // データを利用
}
getUser();

メソッド 読み方 説明
GET ゲット データを取得
POST ポスト データを新しく作成
PUT プット 既存データを上書き
DELETE デリート データを削除

6. まとめ

内容 説明
非同期処理 時間のかかる処理を待たずに進める
Promise 非同期の結果を扱う仕組み
async / await 非同期処理を簡単に書く文法
API 外部サービスとデータ通信できる
fetch() JavaScript で API を呼び出す関数
